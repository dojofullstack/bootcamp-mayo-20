Semantica

La semantica, es el significado de las cosas, en este caso, de las etiquetas HTML. Las etiquetas HTML, tienen un significado, y su uso correcto, ayuda a que los navegadores y los motores de busqueda, entiendan mejor el contenido de la pagina.

Por ejemplo, la etiqueta <header> se usa para definir el encabezado de una pagina, mientras que la etiqueta <footer> se usa para definir el pie de pagina.

El uso correcto de las etiquetas semanticas, ayuda a mejorar la accesibilidad y la usabilidad de la pagina web.

Etiquetas semanticas

  • <header>
  • <nav>
  • <main>
  • <article>
  • <section>
  • <aside>
  • <footer>
