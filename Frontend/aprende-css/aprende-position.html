CSS Position

La propiedad position en CSS se utiliza para especificar el método de posicionamiento de un elemento en la página. Los valores más comunes son:

  • static: Posicionamiento normal, el elemento se coloca en el flujo del documento.
  • relative: El elemento se posiciona relativo a su posición original.
  • absolute: El elemento se posiciona relativo al primer ancestro posicionado (no estático).
  • fixed: El elemento se posiciona relativo a la ventana del navegador y no se desplaza al hacer scroll.
